# Inicio el parcial

# Documentación del Código HTML

1. Propósito del Código:
El código define una página web estática para un hotel llamado "Hotel Ocaña Imperial". El objetivo principal es presentar información básica del hotel, como sus instalaciones, información de contacto, y un video.

2. Estructura del Documento:
- El encabezado:
muestra el título principal de la página, Incluye una barra de navegación con enlaces internos a diferentes secciones del sitio.

- Navegación: 
muestra una lista de enlaces que permiten navegar a las secciones de Inicio, Sobre Nosotros, y Contacto.

- Sección Principal: 
contiene las secciones clave del contenido de la página como el Inicio que introduce brevemente el propósito del hotel, Sobre Nosotros: que describe la historia y servicios del hotel y las instalaciones que presenta el hotel con un video incluido.

- Secciones de Contenido:
Inicio: Breve descripción del hotel.
Sobre Nosotros: Detalles acerca del hotel y su calidad de servicio.
Instalaciones: Información sobre las instalaciones del hotel y un video promocional que muestra visualmente estas instalaciones.

- Video:
se muestra un video en formato .mp4 se encuentra en la sección de instalaciones, mostrando aspectos visuales del hotel.

- Pie de Página (Footer):
Proporciona la información de contacto del hotel e incluye los derechos de autor de la página.
